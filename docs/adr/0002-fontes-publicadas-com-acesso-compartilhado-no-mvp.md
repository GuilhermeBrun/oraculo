# Fontes publicadas tem acesso compartilhado no MVP

No MVP, fontes publicadas no Oraculo podem ser consultadas por todos os usuarios habilitados no sistema, sem controle granular por usuario ou grupo. A decisao simplifica o produto inicial e evita desenhar regras de autorizacao antes de existir uma necessidade concreta, aceitando que documentos restritos nao devem ser publicados nessa base compartilhada.
